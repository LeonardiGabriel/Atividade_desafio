O que são Clients?

Inicio

O client é quem inicia a conexão client-servidor, sendo o client mais conhecido, o navegador.

Ao acessar o navegador, o client envia ao servidor, uma requisição de processamento das informações ao local que o site está hospedado, iniciando assim a conexão de informações com o servidor.

O que são Servers?

Inicio

O servidor é o recurso dentro do sistema que processa as aplicações enviadas pelo client, podendo ser um ou mais computadores conectados simultaneamente em rede, podendo armazenar arquivos de mídia ou aplicações.

O que é um HTML?

Inicio

O HTML (Hypertext markup language) é uma ferramenta de criação de paginas da web focadas em texto, sendo assim, a maioria das páginas são criadas a partir do uso desta ferramenta, apesar de não ser considerada propriamente uma linguagem de programação.
Existem também ferramentas para modificação dos textos, como a "p", utilizada para criação de parágrafos, "style" para mudanças de tamanho, cor e fonte do texto, a "h" utilizada para títulos, a mark utilizada para destacar um texto, entre outras ferramentas não tão interessantes

Entendendo a diferença entre listas ordenadas e não ordenadas

Inicio

Apesar de ambas serem listas, temos diferenças entre elas. A lista ordenada retorna os itens com suas devidas numerações seguindo um padrao (1,2,3,etc.,), já a lista não ordenada, retorna os itens sem um valor especificado, no lugar dos números, temos simbolos que precedem os itens da lista.
